Proyectos realizados

Sistema de documentos con comentarios y permisos — 2025

Desarrollo de un sistema con control de acceso por usuario para comentar documentos. Implementé CRUD para documentos y comentarios, permisos por documento para autorizar quién puede comentar y la visualización del conteo de comentarios.

  • Stack: React + Vite, Express, MongoDB, Tailwind.
  • Enfoque en relaciones entre tablas y visibilidad condicional.
  • Validación de permisos y controles de autorización.
Aplicación móvil educativa (login por roles) — 2025

App móvil con autenticación diferenciada por roles: alumnos, maestros y administrador. Los alumnos consultan materias, horarios, calificaciones y tareas; los maestros gestionan cursos y notas; y el administrador controla usuarios y matrículas.

  • Stack: Kotlin + Firebase.
  • Enfoque en UX por rol y seguridad en accesos.
Plataforma web de capacitaciones — DUX — 2024

Plataforma para ofrecer y gestionar capacitaciones en línea con contenidos interactivos, ejercicios y recursos digitales. Permite mantener presencia constante y ampliar alcance de usuarios.

  • Stack: React con JSX + Firebase.
  • Diseño dinámico, adaptable y enfocado en la experiencia de aprendizaje.
Dashboard de métricas empresariales — 2023

Creación de un dashboard interactivo para visualizar métricas clave de negocio en tiempo real, mejorando la toma de decisiones y la comunicación interna.

  • Stack: Python (Flask), PostgreSQL, Chart.js.
  • Implementación de filtros dinámicos y reportes descargables.
Gestor de inventario para PyME — 2023

Desarrollo de un sistema sencillo de inventario para pequeñas empresas, permitiendo registrar productos, controlar existencias y generar alertas automáticas.

  • Stack: Java + MySQL.
  • Diseñado para ser intuitivo y fácil de usar por personal no técnico.
